# ファイル監視はnotify + ポーリングフォールバックで行う

ワーキングフォルダの変更PDF検知に、Rustの `notify` クレートを使用する。Windowsでは `ReadDirectoryChangesW` API を使うため、Samba（ネットワークドライブ）でも動作する場合が多い。ただしSambaの設定によってはイベントが届かないケースがあるため、`notify` の初期化失敗時またはイベントが一定時間途絶えた場合はポーリング（数秒間隔）に自動フォールバックする。

## Consequences

Samba上のリアルタイム検知は環境依存があるため、フォールバック発生時はUIにインジケーターを表示してユーザーに通知する。
