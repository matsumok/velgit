# 差分画像はTauriカスタムプロトコル経由でReactに渡す

Rustで生成した差分オーバーレイ画像をReactに渡す方法として、一時ファイルに書き出してTauri 2のカスタムプロトコル経由で配信する。ReactはURLを受け取り `<img src="...">` として表示する。

300DPI A3画像（約3500×5000px）はPNGで数MBになるため、base64でIPCを通すとシリアライズのオーバーヘッドが大きい。ファイルパス（文字列）のみをIPCで渡すことでこれを回避する。

## Consequences

一時ファイルはアプリ終了時またはセッション中に適切にクリーンアップする必要がある。
