# ビジネスロジックはRust側に集約する（Rustヘビー）

TauriのRust/React責務分担として、コミット・差分計算・ファイル操作・SQLiteクエリ・メタデータ管理などすべてのビジネスロジックをRust側に置く。ReactはUIの描画とユーザー入力の受け取りのみを担い、Rustから受け取る「表示用に整形済みのデータ」を表示する。

## Consequences

TauriコマンドのAPIは「ReactがUIを描画するために必要な形」に整形されたデータを返す責務を持つ。Reactにビジネスロジックが漏れ出さないよう、invoke呼び出しの戻り値は表示用DTOとして設計する。
